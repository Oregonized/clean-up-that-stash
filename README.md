Clean Up That Stash is a RuneLite plugin that visually categorizes inventory and bank items to help players organize and clean up their stash.

## Features
- Inventory overlay with item categorization
- Bank overlay with item categorization
- Supports all current and future OSRS items
- Configurable overlays
- Visual-only (no automation or item movement)

## Categories
- Melee
- Ranged
- Magic
- Potions
- Skilling
- Quest items

Items are categorized using item properties such as name, equipment slot, and tradeability.

## Compliance
- No automation
- No clicks or item movement
- Overlay-only visuals
- Fully compliant with RuneLite Plugin Hub rules
